# Jessica Nails — Landing

Landing page estática para Jessica Nails Studio. HTML + CSS servido con Nginx.

## Stack

- HTML + CSS
- Nginx
- Docker

## Correr en local

```bash
docker compose up --build -d
```

Abre http://localhost

## Deploy

```bash
git pull && docker compose up --build -d
```

## Dominio

`shop.prittor.com` / `tienda.prittor.com`
